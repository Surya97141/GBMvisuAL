📋 Problem Statement
Given n jobs, each with a start time, end time, and profit, find the maximum total profit from a set of non-overlapping jobs (jobs cannot share time — if one ends at time t, the next must start at t or later).

Approach: Sort by end time → binary search for last compatible job → DP.

dp[0]=0; // base: no jobs
for(int i=1; i<=n; i++) {
  int p=lastCompat(ends,
            jobs[i].start); // binary search
  int take=jobs[i].profit+dp[p];
  int skip=dp[i-1];
  dp[i]=max(take, skip);
}
return dp[n];
