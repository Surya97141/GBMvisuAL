03 How it works

Built different.

Pure HTML, CSS, and JavaScript. No frameworks, no dependencies. Every visualization runs entirely in the browser.

1
Pick an algorithm

Choose from 47 algorithms across 9 categories — arrays, graphs, DP, strings, trees, math and geometry.

2
Set your input

Enter custom inputs or use presets. Each visualizer accepts real data so you can test your own cases.

3
Step through it

Use play, pause, and step controls. Watch pointers move, tables fill, trees expand — one operation at a time.

4
Read the code

Each visualization ships with annotated source code. The logic highlighted in sync with the animation.
