📋 Problem Statement
Given an undirected graph and two nodes S and T, determine if a path exists from S to T.

Algorithm: DFS from S. Mark each node visited as we go. If we ever reach T → path exists. If DFS exhausts all reachable nodes without finding T → no path.

Key insight: DFS explores as deep as possible before backtracking. Every node reachable from S will eventually be visited.

bool hasPath(int s, int t, adj) {
  visited.clear(); // mark none visited
  return dfs(s, t);
}
bool dfs(int u, int t) {
  if(u == t) return true; // reached!
  visited[u] = true;
  for(int v : adj[u]){
    if(!visited[v])
      if(dfs(v, t)) return true;
  }
  return false; // dead end, backtrack
}
