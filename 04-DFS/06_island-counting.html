📋 Problem Statement
Given a 2D grid of '1's (land) and '0's (water), count the number of islands. An island is a group of horizontally or vertically adjacent land cells surrounded by water or the grid boundary.

Approach: Scan every cell. When we find an unvisited '1', increment island count and DFS in all 4 directions to sink the entire island (mark visited), so it won't be counted again.

int numIslands(
    vector<vector<char>>& grid) {
  int R=grid.size(),C=grid[0].size();
  int islands = 0;
  for(int i=0; i<R; i++)
   for(int j=0; j<C; j++)
    if(grid[i][j]=='1') {
      islands++;            // new island!
      dfs(grid,i,j,R,C);
    }
  return islands;
}
void dfs(grid,r,c,R,C) {
  if(r<0||r>=R||c<0||c>=C
     ||grid[r][c]!='1') return;
  grid[r][c] = '0'; // sink / mark visited
  dfs(r-1,c); dfs(r+1,c);
  dfs(r,c-1); dfs(r,c+1);
}
