📋 Problem Statement
Given an undirected graph with n nodes and some edges, find the number of connected components — groups of nodes where every node can reach every other node in the same group.

Approach: For each unvisited node, launch a DFS that marks all reachable nodes as belonging to the same component. Count how many times we launch a fresh DFS.

void dfs(int u, adj, vis) {
  vis[u] = true;
  for (int v : adj[u])
    if (!vis[v])
      dfs(v, adj, vis);
}  // backtrack
