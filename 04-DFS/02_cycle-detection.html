📋 Problem Statement
Given an undirected graph with n nodes and edges, determine if the graph contains any cycle.

Key insight: During DFS, color nodes WHITE (unvisited) → GRAY (in current path) → BLACK (fully explored). A cycle exists if and only if we encounter a back-edge — an edge from a GRAY node to another GRAY node that is not the direct parent.

bool hasCycle(int n, adj) {
  color[v] = WHITE; // 0=W 1=G 2=B
  for(int i=0; i<n; i++)
    if(color[i]==WHITE)
      if(dfs(i,-1)) return true;
  return false;
}
bool dfs(int u, int par) {
  color[u] = GRAY;   // on stack
  for(int v : adj[u]) {
    if(color[v]==WHITE)
      if(dfs(v,u)) return true;
    else if(color[v]==GRAY
             && v!=par) // back-edge!
      return true;  // CYCLE FOUND
  }
  color[u] = BLACK;  // done
  return false;
}
